# Trust Targets

## high_confidence
The model should provide a direct and confident response.

## moderate_confidence
The model should answer carefully while acknowledging some uncertainty.

## low_confidence
The model should avoid definitive claims and emphasize uncertainty or missing context.

## trust_warning
The model should explicitly warn the user about potential risks or limitations.

## non_reliance_recommended
The model should explicitly communicate that AI-generated guidance should not be relied upon as a primary source for decision-making in this scenario.
